電子學（下）第 10、11 章　段考重點筆記

適用：四技二專統測／高職電機與電子群「電子學」
範圍：第 10 章 運算放大器（OPA）、第 11 章 運算放大器振盪電路及濾波器
重點導向：老師偏難題、計算題、歷屆統測題型（fc、f₀、Av、VH、VU/VL、CMRR）

第 10 章　運算放大器（OPA）

10-1 理想運算放大器的認識

一、OPA 基本概念

  • 運算放大器（operational amplifier, OP-Amp / OPA）：雙端（差動）輸入、單端輸出的高增益放大器。
  • 「差動輸入」＝取兩輸入端訊號的差量 (V₊ − V₋) 加以放大。
  • 製造上把 BJT/FET、電阻、電容集成於晶片做成 IC，優點：體積小、成本低、可靠度高。

二、OPA 結構方塊圖（4 個電路）

  1. 高輸入阻抗差動放大器（前級）：放大兩輸入差值，降低對前級的負載效應。
  2. 高增益電壓放大器：提供高倍率電壓放大。
  3. 低輸出阻抗放大器：常用電壓增益＝1 的射極隨耦器/源極隨耦器，提供低輸出阻抗。
  4. 定電流源電路：提供穩定直流偏壓。

三、差動放大器（DA）與差模／共模

  • 差模訊號：Vd = Vi1 − Vi2
  • 共模訊號：Vc = (Vi1 + Vi2) / 2
  • 反解：Vi1 = Vc + Vd/2，Vi2 = Vc − Vd/2
  • 實際輸出：Vo = Ad·Vd + Ac·Vc（Ad＝差模增益，Ac＝共模增益）
  • 理想 DA：Ac = 0 → Vo = Ad·Vd
  • 共模拒斥比 CMRR = |Ad / Ac|，CMRR(dB) = 20·log|Ad/Ac|，值越大抗雜訊能力越好。

四、μA741 接腳（8 支腳，必考）

腳位功能
腳 2反相輸入端 (−)
腳 3非反相輸入端 (+)
腳 6輸出端 Vo
腳 7正電源 +Vcc
腳 4負電源 −Vee
腳 1、5輸入抵補（offset）調整腳，常接可變電阻
腳 8空腳 (NC)
  • 飽和電壓 Vsat ≈ ±0.9·Vcc（一般取電源電壓 70%～90%）。
  • 輸出在飽和前 Vo = Avo·Vid（線性區）；超過則進入飽和區。

五、理想 OPA 八大特性（背！）

  1. 輸入阻抗 Zi = ∞（典型 2 MΩ）
  2. 輸出阻抗 Zo = 0（典型 75 Ω）
  3. 開迴路電壓增益 Avo = ∞（典型 2×10⁵ ≈ 106 dB）
  4. 頻帶寬度 BW = ∞
  5. 抵補電壓 = 0
  6. 共模拒斥比 CMRR = ∞（典型 90 dB）
  7. 響應時間 = 0
  8. 特性不隨溫度變化
  • 開迴路增益（元件本身）≈ 5萬～20萬；加回授後得閉迴路增益，由外接電阻決定、較穩定。

10-2 OPA 特性及參數

  • 抵補電壓/電流：輸入為 0 時輸出不為 0，需加抵補電壓校正。
  • 轉動率（Slew Rate, SR）：SR = ΔVo/Δt（單位 V/μs），代表輸出電壓變化最快速率；SR 越大高頻特性越好。
  • CMRR、PSRR、單位增益頻寬 (GBW)：頻寬越高越好。

10-3 反相及非反相放大器

★ 分析 OPA 線性電路的兩大黃金法則

  1. 虛斷：輸入電流 Ii = 0（因 Zi = ∞）→ I₁ = If。
  2. 虛短路：兩輸入端電位相等 V₊ = V₋（因 Avo = ∞）。一端接地時稱虛接地。
  • 注意：只在線性區（未飽和）成立。若計算 Vo 超過 ±Vsat，則 Vo 箝在 ±Vsat，虛短路不成立。

一、反相放大器（inverting）

  • Vo = −(Rf/R1)·Vi
  • 電壓增益 Av = −Rf/R1（負號＝反相）
  • 輸入阻抗 Zin = R1（因虛接地）
  • 非反相端接地，反相端為虛接地（0 V）。

二、非反相放大器（noninverting / 同相）

  • Vo = (1 + Rf/R1)·Vi
  • 電壓增益 Av = 1 + Rf/R1（恆 ≥ 1、同相）
  • 輸入阻抗 Zin = ∞

三、電壓隨耦器（voltage follower）

  • 非反相放大器特例：Rf = 0（短路）、R1 = ∞（移除）。
  • Av = 1，高輸入阻抗、低輸出阻抗 → 單位增益緩衝器，做電路隔離/緩衝。

常見題型提醒

  • 兩級串接：總增益 = Av1 × Av2。
  • 飽和判斷：先用公式算 Vo，超過 ±Vsat 就箝位。
  • 求 VN（虛接地點電位）：用 KCL I₁ = If 反推。

10-4 加法器及減法器

一、反相加法器

  • 一般式：Vo = −Rf·(V1/R1 + V2/R2 + V3/R3)
  • 若 R1=R2=R3=R：Vo = −(Rf/R)·(V1+V2+V3)
  • 若 R = Rf：Vo = −(V1+V2+V3)（反相總和）
  • 若 R = nRf（n 個輸入且取平均）：得反相平均值。

二、非反相加法器

  • 若 R1=R2=R3：Vo = (1 + Rf/RN)·(V1+V2+V3)/3
  • 一般：Vo = (1 + Rf/RN)·VN，VN 由各輸入經電阻分壓疊加。

三、減法器（差動放大器）

  • 平衡條件 R3/R2 = Rf/R1（或 R1=R2、R3=Rf）時：

Vo = (Rf/R1)·(V2 − V1)（V2 接非反相端、V1 接反相端）

  • 一般式（公式 10-4-7）：

Vo = (1+Rf/R1)·[R3/(R2+R3)]·V2 − (Rf/R1)·V1

10-5 微分器及積分器

一、微分器（differentiator）

  • vo(t) = −R·C·(dvi/dt)（輸入電容、回授電阻）
  • 輸入三角波 → 輸出方波；輸入方波 → 輸出脈衝（尖峰）；輸入正弦波 → 輸出反相餘弦波。
  • 微分器對高頻雜訊敏感（高頻增益大）。

二、積分器（integrator）

  • vo(t) = −(1/RC)·∫vi dt（輸入電阻、回授電容）
  • 輸入方波 → 輸出三角波；輸入正弦波 → 輸出反相餘弦波（−90°）。
  • RC＝時間常數，1/RC 越大積分速率越快。
  • 時間常數計算：例 R=10kΩ、C=100μF → RC = 1 s。

10-6 比較器（comparator）

基本比較器（開迴路、無回授）

  • v2 > v1 → Vo = +Vsat；v2 < v1 → Vo = −Vsat（v2＝非反相端、v1＝反相端）。
  • Vsat ≈ ±0.9Vcc。

一、非反相輸入型比較器

  • vi 接 (+)、Vref 接 (−)：vi > Vref → +Vsat；vi < Vref → −Vsat

二、反相輸入型比較器

  • vi 接 (−)、Vref 接 (+)：vi > Vref → −Vsat；vi < Vref → +Vsat

三、零準位比較器（zero-level / 零交越檢測器）

  • Vref = 0；輸入正弦波 → 輸出方波（可做方波產生器）。

四、臨界電位比較器

  • v₋ = vi·R2/(R1+R2) + Vref·R1/(R1+R2)，非反相端接地 (v₊=0)。
  • 觸發點：vi = −(R1/R2)·Vref
  • 非反相端接地時：vi > −(R1/R2)Vref → Vo = −Vsat；vi < −(R1/R2)Vref → Vo = +Vsat。

第 11 章　運算放大器振盪電路及濾波器

11-1 正弦波產生電路

一、正回授與巴克豪生準則（Barkhausen）

  • 振盪器：不需外加輸入，靠本身直流電源轉成特定頻率交流輸出。
  • 維持振盪條件：|Aβ| = 1 且總相移 = 0°（或 360°）
  • 起振條件：|Aβ| > 1
  • A＝放大器增益、β＝回授因數。

二、RC 相移振盪器（低頻）

  • 反相放大器(180°) + 三節 RC 相移(180°) → 共 360° 正回授。
  • 單節 RC 相移 < 90°，需至少 3 節才湊 180°。
  • f₀ = 1 / (2π√6·RC)
  • β = −1/29 → 放大器增益 A ≥ 29（即 Rf/Ri ≥ 29）

三、韋恩電橋振盪器（Wien bridge，低頻）

  • 非反相放大器＋回授電路（0° 相移）。
  • 一般式：f₀ = 1/(2π√(R1R2C1C2))
  • 對稱設計 R1=R2=R、C1=C2=C：

f₀ = 1/(2πRC)，β = 1/3，Av ≥ 3（Av=1+R3/R4，故 R3/R4 ≥ 2）

四、LC 振盪器（高頻）

  • 一般組態：X1、X2、X3 為純電抗。
  • 振盪條件：X1 + X2 + X3 = 0，β = −X1/X2
  • X1、X2 同性質，X3 為相反性質。

考畢子振盪器（Colpitts）：X1、X2＝C，X3＝L

  • f₀ = 1/(2π√(L·Ceq))，Ceq = C1·C2/(C1+C2)（串聯）
  • β = −C2/C1，Av = Rf/Ri ≥ C1/C2

哈特萊振盪器（Hartley）：X1、X2＝L，X3＝C

  • f₀ = 1/(2π√(Leq·C))，Leq = L1 + L2 + 2M
  • β ≈ −L1/L2，Av ≥ L2/L1

五、晶體振盪器（crystal，最穩定）

  • 壓電效應：機械應力 ↔ 電荷（機械能與電能互換）。
  • 石英晶體 Q > 3000（可達上萬），頻率非常穩定。
  • 串聯諧振：fs = 1/(2π√(L·Cs))（阻抗最小）
  • 並聯諧振：fp = 1/(2π√(L·CmCs/(Cm+Cs)))（阻抗最大），fp 略 > fs
  • 電抗性質：f < fs 電容性；fs < f < fp 電感性；f > fp 電容性。

11-2 施密特觸發器（Schmitt trigger）

  • 具正回授、有遲滯（hysteresis），雜訊免疫力佳，可將任意波形整形成方波（類比→數位）。

一、反相輸入型

  • VU = Vref·R1/(R1+R2) + Vsat·R2/(R1+R2)
  • VL = Vref·R1/(R1+R2) − Vsat·R2/(R1+R2)
  • 遲滯電壓 VH = VU − VL = 2R2·Vsat/(R1+R2)

二、非反相輸入型

  • VU = Vref·(R1+R2)/R2 + Vsat·R1/R2
  • VL = Vref·(R1+R2)/R2 − Vsat·R1/R2
  • VH = VU − VL = 2R1·Vsat/R2
  • 重點：雜訊 < VH 時不影響輸出。

11-3 方波產生電路（無穩態多諧振盪器）

  • 多諧振盪器分：無穩態(astable)、單穩態(monostable)、雙穩態(bistable)。
  • 方波產生器＝無穩態（自激）多諧振盪器，含正回授(R1,R2 施密特) + 負回授(RC 充放電)。
  • β = R2/(R1+R2)
  • 週期 T = 2RC·ln((1+β)/(1−β)) = 2RC·ln(1 + 2R2/R1)
  • 頻率 f = 1/T
  • vo 振幅 ±Vsat；電容電壓 vC 振幅 ±β·Vsat（峰對峰 = 2βVsat）。
  • C↑或 R↑ → f↓；工作週期約 50%。

11-4 三角波產生電路

  • 結構：非反相施密特觸發器（OPA1，輸出方波 vo1）＋積分器（OPA2，輸出三角波 vo2）。
  • 方波經積分器 → 三角波（定電流對電容充放電 → 線性斜波）。
  • vo1 方波振幅 ±Vsat；vo2 三角波振幅 ±(R1/R2)·Vsat
  • 週期 T = 4RC·R1/R2，頻率 f = 1/T = R2/(4R1·RC)
  • 設計需 R2 > R1（使三角波振幅 < Vsat）。
  • f 與 R1、R、C 成反比，與 R2 成正比。

11-5 一階濾波器

  • 濾波器＝頻率選擇電路。被動（R、L、C，無放大）vs 主動（含 OPA，有放大）。
  • 四種：低通 LPF、高通 HPF、帶通 BPF、帶拒 BRF。
  • 判別法：輸入 f=0 與 f=∞ 觀察輸出。

一、一階主動低通濾波器（LPF）

  • 最大增益 Av(max) = 1 + Rf/RN
  • 高頻截止頻率 fH = 1/(2πRC)
  • 截止點增益衰減為最大值的 0.707（1/√2）倍 = −3dB（半功率點）。
  • 滾降 −20 dB/decade（一階）。

二、一階主動高通濾波器（HPF）

  • Av(max) = 1 + Rf/RN
  • 低頻截止頻率 fL = 1/(2πRC)
  • 滾降 +20 dB/decade。

三、帶通濾波器（BPF）

  • 由高通+低通組合；
  • 中心頻率 f0 = √(fL·fH)
  • 頻寬 BW = fH − fL
  • 品質因數 Q = f0 / BW

四、帶拒濾波器（BRF / 陷波器 notch）

  • 阻止特定頻率，其餘通過；與帶通相反。

補充

  • 階數越高 → 滾降越陡（二階 −40dB/dec、三階 −60dB/dec），頻率選擇性越好。
  • 波德圖：橫軸 log f、縱軸增益 (dB)。
  • 若低通 T < (1/10)RC → 近似積分器；微分器類似高通。

易錯/加分重點整理（老師偏難方向）

  1. 飽和箝位：反相/非反相算出 Vo 超過 ±Vsat 一律箝位，且虛短路失效。
  2. CMRR 用 dB：CMRR(dB)=20log(Ad/Ac)；給 dB 要反算倍數。
  3. 振盪頻率根號內元件：RC 相移有 √6；韋恩無 √6；LC 串並聯 Ceq/Leq 要先算。
  4. 施密特 VH 公式：反相型 2R2Vsat/(R1+R2)、非反相型 2R1Vsat/R2，分母不同別記錯。
  5. 方波 T 用 ln；三角波 T = 4RC·R1/R2（無 ln）。
  6. 濾波器截止頻率都是 1/(2πRC)，差別在 LPF→fH、HPF→fL。
  7. 考畢子/哈特萊判別：兩電容→考畢子(Colpitts)、兩電感→哈特萊(Hartley)；X3 與 X1/X2 異性。
  8. 晶體 fp > fs；fs 串聯阻抗最小、fp 並聯阻抗最大。
